巴哈自製遊戲公會活動

-遊戲題目:
「既然你誠心誠意的請教了，我就大發慈悲的告訴你。為了防止耶穌像被破壞，為了
守護聖誕的和平，貫徹愛與真實的蛋糕，可愛又迷人的彩色禮物。蘿莉。正太。穿梭
於大哥哥大姊姊之間要禮物的小朋友。娃娃、玩具的房間再等著我們。」

-遊戲前言:
傳說有個玩具屋，會一直出現許許多多的玩具，
所以許多死小孩爭相要擠進那個玩具屋，
在玩具屋前面有個歷史悠久耶穌像，
常常被小孩子破壞，於是你化身禮物使者，
要把玩具屋內的玩具丟離耶穌像，
去防止耶穌像被破壞，於是遊戲就這樣展開了…


-遊戲機制:
1.耶穌像:
我們要守護的基地台 分別有血條量與魔力值
小朋友經過就會損血
成功讓小朋友拿到禮物便增加魔力值
增加魔力值就能加速禮物產生的速度

2.禮物使者:
大哥哥大姊姊們也就是主角們
不同的人哥哥姊姊有不同的技能去敵擋小朋友們
其中可以有修繕耶穌像幫耶穌像補血的

3.小朋友: 敵人們
他們
隨時間數量會越來越多
速度也會越來越快
也會產生比較貪心的小朋友

4.蛋糕:
小朋友手中會拿的東西
當他們拿走禮物便會放下手中的蛋糕
集滿越多蛋糕 就能招募越多禮物使者
